# Prédiction de la réussite scolaire avec un perceptron multicouche

## Description
Projet académique réalisé dans le cadre du master MIAGE.
Ce projet vise à prédire la réussite scolaire des élèves
à partir de facteurs socio-éducatifs.

## Objectifs
- Comprendre le fonctionnement d’un perceptron multicouche
- Appliquer les bases de l’intelligence artificielle
- Manipuler des données éducatives simples

## Technologies utilisées
- Python
- Keras / TensorFlow
- Pandas
- NumPy

## Fonctionnalités
- Entraînement d’un modèle de perceptron
- Prédiction de la réussite ou de l’échec
- Interface simple de visualisation des résultats

## Niveau
Débutant (projet d’apprentissage)
